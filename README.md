# 2hu8

The **2hu8** is a small, split keyboard meant for playing Touhou Project-type shmups.
Games that use a similar layout should also be playable (e.g. Undertale/Deltarune, other shmups, etc.)

## Layout

The board is split into two halves, both in the shape of the arrow key cluster.

| Half  | Keys                   |
| ----- | ---------------------- |
| Left  | Shift, Z, X, Esc, R, C |
| Right | Arrow keys             |

The left half houses the microcontroller, and is connected to the right half with a TRRS cable.

## Firmware

The 2hu8 is powered by QMK.

## Why is it 2hu8 when there's 10 keys?

The original design was to have 2 arrow key cluser-shaped boards for a total of 8 keys,
but then I realized that I forgot to add Esc (for pausing) and R
(in most Touhou games, pressing R in the pause menu quick restarts the run).
